머신러닝이란?

y = f(X)

입력값 X(x1, x2 ..)에 어떤 로직 f를 적용하면 결과값 y가 나옵니다.

입력값 X와 로직 f를 알려줄께, y값을 구해줘

입력값 X와 출력값 y를 알려줄께, f를 좀 알아내줘

간단한 용어 설명

X feature

Y target

f model

학습 : X, y -> f

추론 : f(X') -> y'

Regression vs Classification

Regression : 수치를 예측

Classification : 분류
